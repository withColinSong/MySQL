# 1. 다중 테이블 SELECT
```sql
SELECT 컬럼
FROM `테이블1`
INNER JOIN `테이블`
    ON 데이터 연계 조건
```

# 2. 단순 관계

## 2.1 일대일 관계
- 단순한 일대일 관계
ex) 글 작성자와 이메일 주소의 관계

## 2.2 다대일 관계
- 글 작성자는 한명이지만, 한 작성자는 여러 글을 작성할 수 있다.
- 공간 낭비될 우려를 위해 데이터를 두 테이블로 나누고 `join쿼리`로 해결
